# حصن المسلم — تطبيق الأذكار 📿

تطبيق ويب عصري وخفيف للأذكار والأدعية من القرآن الكريم والسنة النبوية، يعمل كتطبيق متصفح PWA قابل للتثبيت ويدعم العمل بدون اتصال.

## ✨ الجديد في الإصدار 3.0

- 🪶 **خفيف جداً**: قشرة `index.html` صارت ~9 كيلوبايت بدل 200+ (تحسّن ~96%)
- 📦 **بيانات مجزّأة**: كل قسم في ملف JSON منفصل تحت `assets/data/categories/` (132 ملفاً + `index.json` للفهرسة)
- ⚡ **تحميل عند الطلب (Lazy load)**: لا يحمَّل القسم إلا عند فتحه — لا يثقل التشغيل الأولي
- 🌗 **سمة فاتحة/داكنة**: تبديل يدوي + اتباع تفضيل النظام
- 📲 **PWA كامل**:
  - تثبيت من المتصفح
  - `manifest.webmanifest` + `sw.js`
  - تخزين القشرة وبيانات الأذكار للعمل أوفلاين
  - تخزين صوتي اختياري لكل ذكر
- 🎨 **واجهة جديدة**: خط Tajawal + Amiri Quran، بطاقات نظيفة، ذكر عشوائي بارز، تبويبات قابلة للتمرير
- 🔍 **بحث ذكي تدريجي**: نتائج تظهر فور تحميل كل قسم
- ❤️ **مفضلة فعلية** + **أذكاري المخصصة**: مع حفظ في `localStorage`
- 🛡️ **بدون اعتمادات بناء**: يعمل بفتح `index.html` مباشرة أو رفعه على أي استضافة ثابتة

## 🚀 التشغيل محلياً

أبسط طريقة:

```bash
cd GT_HISNMUSLIM-main
python3 -m http.server 8000
# ثم افتح http://127.0.0.1:8000
```

> ⚠️ ميزات Service Worker / التثبيت تتطلب تشغيل عبر `http://localhost` أو `https://`، ولا تعمل عند فتح الملف مباشرة من `file://`.

## 📱 كيفية الاستخدام

1. افتح التطبيق في المتصفح
2. اختر القسم من شريط التبويبات أو اضغط رؤوس الأقسام في عرض "الكل"
3. اضغط ♡ لإضافة الذكر إلى المفضلة
4. اضغط 📋 لنسخ الذكر، و ⬇️ بجوار المشغل لتخزين الصوت أوفلاين
5. اضغط زر **تثبيت** في الرأس لتثبيت التطبيق على جهازك (يظهر تلقائياً عندما يدعمه المتصفح)
6. استخدم 🌗 لتبديل الوضع الداكن/الفاتح
7. من القائمة (☰): إضافة ذكر مخصص، أو إدارة التخزين الصوتي

## 🛠️ التقنيات

- HTML5 + CSS3 (متغيرات CSS + Logical Properties للدعم RTL)
- JavaScript (ES6+) — وحدات IIFE معزولة، بدون أي اعتمادات
- Service Worker API + Cache API
- Web Notifications API (اختياري)
- Web Share API (اختياري)
- خط Tajawal + Amiri Quran من Google Fonts
